OpenClaw exposes first-class agent tools for browser, canvas, nodes, and cron. The tools are typed, no shelling, and the
agent should rely on them directly. Note the cron job is the Agent run, not the command execution. It can be using executables, but not necessary

The skills that use the tools are installed in the C:\Users\uberdev\AppData\Roaming\npm\node_modules\openclaw\skills

Tools are exposed in two parallel channels:

System prompt text: a human-readable list + guidance.
Tool schema: the structured function definitions sent to the model API.
That means the agent sees both “what tools exist” and “how to call them.” If a tool doesn’t appear in the system prompt or the schema, 
the model cannot call it.
